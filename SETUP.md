# Quick Setup Guide

## 🚀 Get Started in 3 Steps

### Step 1: Open the Files
Download all the website files to your computer and open them in a text editor:
- **Recommended editors:** VS Code, Sublime Text, or Notepad++
- Open `index.html`, `styles.css`, and `script.js`

### Step 2: Customize Your Content
Use Find & Replace (Ctrl+F or Cmd+F) in `index.html` to update:
1. `[Your Name]` → Your actual name
2. `[Your Profession]` → Your job title
3. `[Your Institution/Company]` → Where you work/study
4. `your.email@example.com` → Your real email
5. `yourusername` → Your GitHub/LinkedIn username

### Step 3: Add Your Images
1. Create a folder called `images` next to your HTML file
2. Add these images:
   - `profile.jpg` - Your headshot (500x500px or larger, square)
   - `project1.jpg`, `project2.jpg`, `project3.jpg` - Project screenshots
   - Or update the image filenames in `index.html` to match your files

### Step 4: Test Locally
Simply double-click `index.html` to open it in your browser and see your website!

### Step 5: Customize Colors (Optional)
Open `styles.css` and change the colors at the top:
```css
--primary-color: #2c5f7c;      /* Change this hex code */
--secondary-color: #e67e22;    /* And this one */
```

Use a color picker like [Coolors.co](https://coolors.co/) to choose colors.

## 📁 File Structure

Make sure your files are organized like this:
```
my-website/
├── index.html
├── styles.css
├── script.js
├── README.md
├── SETUP.md (this file)
└── images/
    ├── profile.jpg
    ├── project1.jpg
    ├── project2.jpg
    └── project3.jpg
```

## 🌐 Deploy Your Website

### Easiest Option: Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your website folder
3. Your site is live!

### Free Option: GitHub Pages
1. Create account at [github.com](https://github.com)
2. Create repository named `yourusername.github.io`
3. Upload files
4. Site live at `https://yourusername.github.io`
